# KP Liberation – ACE Arsenal / BI Arsenal via ACE Interact

> This setup is designed to support the [Vehicle Inventory System Mod](https://steamcommunity.com/sharedfiles/filedetails/?id=2161671937).

Using this method, the arsenal can only be accessed through the crate using the ACE interaction menu. The standard scroll menu option for accessing the arsenal can be disabled by commenting out the arsenal code in:

`Missionframework/functions/fn_addActionsPlayer.sqf`

This method also improves compatibility with ACE CSW, as it allows ammo to be loaded directly from the crate without the need to unload it first.
